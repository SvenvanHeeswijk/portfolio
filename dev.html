DEVELEPMONT

De eerste POC die ik heb gemaakt is een Rock Paper Scissor game. Deze is gemaakt met de hulp van Javascript, voordat ik begon met de opdracht had ik nog niet veel ervaring met Javascript. Ik had al een basis van C# uit het eerste semester, met deze basiskennis had ik al wel een mooie basis voor Javascript. Wat ook een voordel was is dat al een C# Rock Paper Scissor game had gemaakt dus ik wist de basis van de opdracht.

Ik heb via de volgende volgorde de POC opgebouwd
1. Je maakt een keuze uit 3 afbeeldingen
2. De computer maakt zijn keuze en beeld het af
3. De computer laat zien wie heeft gewonnen.

De keuze van de speler spreekt voor zich hoe die geselecteerd wordt, echter is dat bij cumputer wat anders. Die kan niet zo bedenken dat hij een steen moet kiezen. Ik heb hierbij gekozen om een array te maken met de 5 keuzes erin. Ik heb gekozen voor dubbele om zo de willekeurigheid te verhogen. Vervolgens kiest de computer en willekeurig getal tussen de 0 en 5, met deze keuze heeft de computer een waarde die hij uit de array haalt en daardoor ook een keuze heeft gemaakt.

Vervolgens wordt er gekeken via een IF statement wie er gewonnen heeft. Dit is een redelijk simpel stukje code. Wat ook gebeurd in de IF statement is dat de keuze van de computer een achtergrond kleur krijgt om ze te laten zien wat de keuze ook daadwerkelijk is. Het laatste wat nog gebeurd in de IF statement is de dat er een tekst wordt geplaatst met wie de uiteindelijke winnaar is.

Wat heb ik geleerd?

Tijdens deze opdracht heb ik de basis geleerd van Javascript. Ik heb hierbij vooral geleerd hoe ik mijn gedachte om moet gaan zetten in code. Ook heb ontdekt dat de basis van Javascript redelijk overeen komt met de basis van C#.
